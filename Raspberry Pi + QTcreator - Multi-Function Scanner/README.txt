The Raspberry Pi Linux image was custom-made for the project's needs using Buildroot.

The code was developed in C and C++ using PThreads as the multitasking API, QTCreator was employed to implement the GUI on a 5-inch touchscreen. Additionally, the system integrated a temperature sensor, and a custom device driver for the RGB LED was implemented from scratch.

SQLite was used as an embeedded database in order to store all the stock information.

ThingSpeak COTS was utilized to record all sales performed by the scanner and represent them graphically.

More information can be found on the provided project report.
